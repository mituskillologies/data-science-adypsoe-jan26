# Data Science Training Program - Jan 2026
### Ajeenkya DY Patil School of Engineering, Pune

This repository contains code examples, Jupyter notebooks, and resources from the **Data Science Training Program** conducted at Ajeenkya DY Patil School of Engineering (ADYPSOE) in January 2026.

The materials cover the complete Data Science pipeline, starting from Python programming fundamentals to Machine Learning algorithms and model deployment.

* **13 data_viz_Jan8.ipynb**: Data visualization using Matplotlib/Seaborn.
